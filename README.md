## **How to install and run locally?**

```
$ git clone https://github.com/Krushna-Kulkarni/BookBugs
$ cd BookBugs
$ npm install
$ npm start
```

## **Features**

- Home page
- Product listing page
- Filter and sort products
- Search product
- Individual product page
- Cart Managment
- Wishlist managment
- Address Managment
- Checkout page
- Alerts and Loaders
- Authentication:
  - User Signup
  - User Login
- Fully Responsive

## **Built with**

- HTML5
- Vanilla CSS
- React JS
- React Context API
- React Router v6
